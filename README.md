# 더소피 매일학습

고3 매일학습지 20회차를 학생별로 제출·채점·누적 관리하는 새 웹사이트 프로젝트입니다.

구현 범위:
- 학생 개별 로그인
- 원장 및 선생님 공통 로그인
- 모고·모평/변형 문제 별도 즉시 채점
- 문항별 정오답과 해설
- 두 답안 제출 뒤 괄호형 정답 전체 공개
- 학생별 누적 성적 및 제출·미제출 현황
- 학생 CSV 일괄 등록과 비밀번호 재발급
- 학부모 전달용 PNG 성적표
- 원본 DOCX 20회차 정답·해설 자동 추출
- PostgreSQL 운영 배포 지원

전체 소스는 완성되었으며 Python Flask 서버와 데이터베이스를 사용하는 구조입니다. GitHub Pages 같은 정적 호스팅이 아니라 Render·Railway·Fly.io 등 서버 호스팅에 배포해야 합니다.
