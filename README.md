# 더소피 매일학습

고3 매일학습지 20회차를 학생별로 제출·채점·누적 관리하는 웹사이트 프로젝트입니다.

## 구현 범위

- 학생 개별 로그인
- 원장 및 선생님 공통 로그인
- 모고·모평과 변형 문제 별도 즉시 채점
- 문항별 정오답·정답·해설
- 두 답안 제출 뒤 괄호형 정답 전체 공개
- 학생별 누적 성적 및 제출·미제출 현황
- 학생 CSV 일괄 등록과 비밀번호 재발급
- 학부모 전달용 PNG 성적표
- 원본 DOCX 20회차 정답·해설 자동 추출
- PostgreSQL 운영 배포 지원

## 개인정보 운영 원칙

실제 학생 등록에는 학생 이름, 로그인 아이디, 암호화된 비밀번호와 성적 기록만 사용합니다. 학교·학년·반·연락처·학부모 정보는 입력하지 않습니다. 운영 비밀번호, `.env`, 데이터베이스 파일은 공개 저장소에 올리지 않습니다.

## 전체 소스 펼치기

GitHub 연결을 통해 전체 프로젝트를 압축 패키지로 업로드했습니다. 저장소를 내려받은 뒤 최상단에서 다음 명령을 한 번 실행하면 `app.py`, `templates`, `static`, 배포 설정 파일이 펼쳐집니다.

```bash
python unpack_source.py
```

이 사이트는 로그인과 공용 성적 저장이 필요한 Flask 서버형 앱입니다. GitHub는 소스 보관 장소이며, GitHub Pages만으로는 학생별 로그인과 여러 기기의 성적 공유가 작동하지 않습니다. 실제 공개 주소는 Render·Railway·Fly.io 같은 서버 호스팅에 이 저장소를 연결해 생성합니다.
